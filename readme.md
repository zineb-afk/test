# Login Authentication System

A simple login authentication system built with HTML, CSS, and JavaScript.

## Files

- **index.html** - Welcome page
- **login.html** - Login page
- **home.html** - Home page (after login)
- **About.html** - About page
- **Contact.html** - Contact page
- **login.js** - Login logic
- **index.css** - Styles

## How to Use

1. Open `index.html` in your browser
2. Click "Go to Login" button
3. Enter credentials:
   - Username: `admin`
   - Password: `admin`
4. Click Login button to access the home page

## Features

- Simple form validation
- Navigation between pages
- Responsive design
- Clean UI with gradient backgrounds
